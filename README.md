# Science Course Pathway Planner

Student-facing science course advising website.

## What's included

- Multi-page guided planner instead of one long form
- Grade-specific course choices
- 9th grade: Biology or Honors Biology
- 10th grade: Physical Science, Regular Chemistry, or Honors Chemistry
- 11th grade: RG Drage Technical School option or high-school science route
- 11th/12th grade expanded science options
- 12th grade Organic Chemistry option only
- Nursing/allied health pathway
- Non-science college major pathway
- STEM, science major, career tech, public safety, workforce/undecided pathways
- Junior/senior science double-up option
- Printable/saveable recommendation page

## Upload to GitHub Pages

Upload the files in this ZIP to your repository. The main page is `index.html`.

## Quick edits

Open `index.html` and search for:

- `gradeCourses` to edit available courses
- `goals` to edit student future-path options
- `interests` to edit interest buttons
- `pathways` to edit pathway summaries and advice
